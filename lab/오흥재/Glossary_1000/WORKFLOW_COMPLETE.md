## 💡 LLM 학습 활용 방법

### 1. RAG 기반 검색
```python
# 사용자 질문 예시
user_query = "대항력이 뭔가요?"

# 벡터 DB에서 유사 용어 검색
results = vector_db.search(user_query, top_k=3)

# LLM에게 컨텍스트 제공
context = "\n".join([
    f"용어: {r['term']}\n설명: {r['definition_easy']}"
    for r in results
])

prompt = f"""
다음 용어집 정보를 참고하여 사용자 질문에 답변해주세요:

{context}

사용자 질문: {user_query}

답변 시 주의사항:
- 쉬운 말로 설명
- 예시 포함
- 관련 용어도 함께 안내
"""
```

### 2. Few-shot Learning
```python
# 프롬프트에 몇 개 예시 포함

사용자 질문: {user_query}
"""
```

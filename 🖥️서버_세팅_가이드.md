# front-end 서버

## front 세팅
- node.js 설치 [node.js 다운 url](https://nodejs.org/ko/download)
- 설치시 아무것도 클릭 없이 next 만
- 경로 이동 > `{프로젝트_경로}\front-end\zf_front`
- shell 명령어
```shell
npm install
```

## front 가동
- 경로 이동 > `{프로젝트_경로}\front-end\zf_front`
- shell 명령어
```shell
npm run dev
```

# back-end 서버

## back-end 세팅
```shell

# 환경 설정
conda create -n zf_back python=3.12

# 환경 활성화
conda activate zf_back

# 필수 설치
pip install fastapi uvicorn "pydantic[email]"
pip install mysql-connector-python aiomysql langchain openai
pip install sentence_transformers dotenv asyncpg

```


## back-end 가동
- 경로 이동 > `{프로젝트_경로}\back-end\zip_fit`
```shell
# 실행
uvicorn main:app --reload
```
